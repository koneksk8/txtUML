- Install Arduino SDK from https://www.arduino.cc/download_handler.php
- On Windows, copy the lib/rxtxSerial.dll into C:\Windows\SysWOW64, then add this directory to your PATH
- Under Linux, copy lib/librxtxSerial.so to <JAVA_HOME>/jre/lib/x86_64/ or <JAVA_HOME>/jre/lib/amd64/
- Connect the devices, wait until the drivers are installed
- On Windows, run Eclipse as system administrator
- Run or debug the model
